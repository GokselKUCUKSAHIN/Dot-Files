### 🍎 OS X

- Enable TRIM Support to 3rd Party SSDs.
  ```bash
  sudo trimforce enable
  ```

- Delete All Locale TimeMachine Snapshot files.
  ```bash
  for d in $(tmutil listlocalsnapshotdates | grep "-"); do sudo tmutil deletelocalsnapshots $d; done
  ```
